# Casa Sol — Sample Project

A demo project for planhaus. Fictional young couple designing a ~60m² Mediterranean apartment in Málaga.

## Quick Reference

- **Brief**: `brief.yaml` — warm modern, mid-range budget
- **Layout**: `apartment.yaml` — 2 rooms (living + bedroom)
- **Rooms**: `rooms/living-room.yaml`, `rooms/bedroom.yaml`
- **Registry**: `registry/furniture/` — 8 sample items

## Commands

```
/planhaus:validate living-room
/planhaus:render living-room
/planhaus:position living-room --matrix
/planhaus:review apartment
```
